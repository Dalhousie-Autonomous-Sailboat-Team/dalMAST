Macro repeat.

This macro represents a horizontal repetition construct.

Parameters
[in]countThe number of repetitious calls to macro. Valid values range from 0 to MREPEAT_LIMIT.
[in]macroA binary operation of the form macro(n, data). This macro is expanded by MREPEAT with the current repetition number and the auxiliary data argument.
[in]dataAuxiliary data passed to macro.
Returns
